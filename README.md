# Undoer
A simple mod for Flyout that adds full undo and redo to the Craft Editor! Every edit you make is automatically saved as a snapshot. Press Ctrl+Z to undo and Ctrl+Y to redo, just like you'd expect. Press F7 to cycle through memory sizes (50, 100, 200, or 500 steps). Works with everything from placing parts to tweaking propeller blade twist!
